# OLO

**OLO is an website dedicated for waterloo students so people can find out school events whithin 1 click.**

**Technology**
React, Node, Express, Postgresql, Redis

**Design**
Figma: https://www.figma.com/design/ehucDNnd1U26YYYykvPX4B/Evericons-Icon-Set-by-Iconduck--Community-?t=KLUYBBS9adZGgubK-0

**Features**

- Web scrap school events from instagram so students do not need to check different organization accounts throughout the day to know the most updated information
- Ollama is used to extract event information from the instagram post
- User can filter out the events they want to attend based on conditions
- User can add events to their schedul to preview their day
- User can make connections with people who wants to go to same events
